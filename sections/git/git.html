Основные используемые команды Git

Создание репозитория

git init

git init создаёт пустой репозиторий или повторно инициализируйте существующий [Что это означает?]

Операции с файлами

git add

git add <имя_файла> добавляет указанный файл в индекс
  • если файлов несколько, то они перечисляются через пробел
  • указание точки вместо имени файла добавит все файлы текущей директории и её поддиректорий
  • указание звёздочки вместо имени файла добавит все файлы текущей директории и её поддиректорий
