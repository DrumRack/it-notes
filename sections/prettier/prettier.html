Официальный сайт: Prettier

Установка

npm install --save-dev --save-exact prettier

printWidth: <число> --print-width <число> 80 Максимальное количество символов в строке.
tabWidth: <число> --tab-width <число> 2 Количество пробелов в отступах.
useTabs: <true|false> --use-tabs false (Табуляция будет использоваться для отступов, но Prettier использует пробелы для выравнивания элементов, например, в тернарах. Такое поведение известно как SmartTabs.)
